# smoled

A very smol editor.

## Building

`nasm -fbin smoled.asm -o smoled.com`

or

`tinyasm -f bin smoled.asm -o smoled.com`

## Usage

Run `somled.com`

Cursor keys:

```
Up: ctrl+p
Down: ctrl+n
Left: ctrl+b
Right: ctrl+f
Start of line: ctrl+a
End of line: ctrl+e
Quit: ctrl+q (twice, if dirty)
Save: ctrl+s (always saves to SMOLED.TXT for now)
Delete: ctrl+d (not "Del" key)
```
